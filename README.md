# MultiTabList

微信小程序，多个Tab列表的上下拉刷新方案

> 由于顶部多Tab视图，原生的下拉刷新不适合（用小程序原生下拉交互会很奇怪，更像是全局刷新而不是当前list刷新）

![gif_loading](https://user-images.githubusercontent.com/9360037/40223237-ed087906-5ab4-11e8-8890-e69fa168fb0a.gif)

### 支持功能

- [x]  等间距Tabs展示
- [ ]  顶部多个Tab可滚动展示
- [x]  左右滑动手势切换List
- [x]  网络错误、数据空白占位提示
- [x]  刷新 & 分页懒加载
- [x]  上拉刷新（三个state）
